About the Practice

These works are created using TouchDesigner, a node-based visual programming environment for real-time interactive media. Key techniques include:

  • Feedback loops — visual systems that respond to and modify their own output
  • Audio-reactive parameters — sound analysis driving visual transformations in real time
  • Particle systems — thousands of points moving according to physics-based rules
  • GLSL shaders — custom GPU programs for rendering complex visual effects
